3. Formatting Corrections

Headings

Use H1 for section titles, H2 for subsections, H3 for formulas.

Tables

Use consistent formatting:

| Property | Value | Notes |

Code Blocks

Use triple backticks for:

• HTLC scripts
• Solidity pseudocode
• OP_RETURN examples


Formulas

Use LaTeX consistently:

x \cdot y = k


Lists

Use bullet points for clarity.

Spacing

Ensure:

• 1 blank line between sections
• 0 blank lines inside formulas
• consistent indentation
