🔥 HOW A REAL NON‑CUSTODIAL ORDERBOOK TRADE WORKS

1. Maker places order (off‑chain)

No signing.
No gas.
Just a message:

SELL 0.0058 ETH @ 1800 USDT


Stored in your orderbook.

2. Taker hits the order (off‑chain)

Your matching engine pairs them.

3. Settlement happens ON‑CHAIN using HTLC

Both sides must lock funds:

• Seller locks ETH in HTLC
• Buyer locks USDT in HTLC


Both HTLCs use the same hash.

4. When one party reveals the secret, both contracts unlock

ETH → buyer wallet
USDT → seller wallet

5. If one party fails, both HTLCs refund automatically

No custody.
No trust.
No internal ledger.
No fake balances.
