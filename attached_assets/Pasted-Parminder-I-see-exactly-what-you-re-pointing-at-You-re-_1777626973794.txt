⭐ THE REAL REASON YOUR PAIRS LIST CHANGES EVERY REFRESH

Your Markets page is not static.

✔ Every refresh = new API call

✔ LetsExchange returns a different set of supported pairs depending on:

• liquidity availability
• rate availability
• temporary routing outages
• maintenance on specific chains
• internal routing changes
• rate‑limit fallback
• quote provider rotation
• aggregator backend load


✔ So the list of coins changes constantly

This is why:

• BTC list shows 152 coins, then 717 coins
• BSV list shows 142 coins, then different coins
• ETH list shows different coins
• Some coins appear only sometimes
• Some coins disappear randomly
• Some coins reorder themselves


This is normal for an aggregator‑driven market list.

---

⭐ WHY IT HAPPENS (TECHNICAL DETAILS)

1. LetsExchange does NOT return a fixed list

Their /info/coins endpoint returns:

• only coins currently routable
• only coins with active liquidity
• only coins with available quotes
• only coins with healthy routing paths


If a coin’s liquidity provider is down → coin disappears.
If a new provider comes online → coin appears.

This is why your list jumps from:

• 152 → 717 → 142 → 300+ → 500+
depending on the moment.


---

2. Your frontend builds markets dynamically

Your code does:

fetch /api/markets?quote=BTC


Backend does:

get all coins from LetsExchange
convert price into BTC
filter invalid quotes
return list


So the list is entirely dependent on LE’s live response.

---

3. Cross‑chain routing changes constantly

LE uses:

• internal liquidity
• external liquidity
• multiple providers
• fallback routes
• chain health checks


So the set of “available” coins changes minute‑to‑minute.

---

4. You are showing ALL external markets

You’re not filtering or stabilizing the list.

So you show:

• every coin LE supports
• every coin LE temporarily supports
• every coin LE temporarily drops
• every coin LE re‑adds


This is why the list looks like it’s “breathing”.

---

⭐ WHY THE NUMBERS ARE DIFFERENT BETWEEN BTC / BSV / ETH

Because each quote is:

price = LE_quote(coin → BTC)


If LE cannot compute a quote for that pair:

• coin disappears from BTC list
• but may still appear in ETH list
• or BSV list
• or USD list


This is why:

• BTC list shows 717 coins
• BSV list shows 142 coins
• ETH list shows ~200 coins
• USD list shows different coins


Each quote path is independent.
