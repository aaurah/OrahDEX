⭐ THE WALLET YOU WANT = 1 WALLET, 4 SIGNING ENGINES

Your wallet must support four signing modes, all under one identity:

1. Imported Private Key (raw key)

• User imports a private key or seed phrase
• Stored encrypted with PIN + device key
• Decrypted only in memory
• Works for all chains derived from that key


2. Passkey Wallet (WebAuthn)

• Keypair generated in secure enclave
• Private key never leaves device
• Authentication via FaceID / TouchID / PIN
• Perfect for EVM, Solana, Tron, BTC, BSV, BCH


3. MPC‑style Passkey Recovery

• Passkey + server share
• If user loses device → recover using identity
• No seed phrase needed


4. WalletConnect (external wallets)

• Connect MetaMask, OKX, Trust, imToken, TronLink, etc.
• For users who don’t want to import keys


All four must be unified under one identity and one wallet UI.
