2. Price Engine — Needs Consistent Definition

You describe:

• TWAP
• VWAP
• BSV on-chain feed
• 210-symbol aggregation


But the white paper never defines:

• which source is authoritative
• how conflicts are resolved
• how stale data is handled


Required correction:

The sovereign price engine selects the median of three sources:1. OrahDEX Order Book VWAP
2. OrahDEX TWAP (60s rolling window)
3. BSV on-chain oracle feed
If any source is stale (>30s), it is excluded.

This ensures deterministic pricing.

You describe:

• relayers
• watchtowers
